Detailed Description

template<class T>
class log4cplus::spi::FactoryRegistry< T >

This template class is used as a "Factory Registry". Objects are "entered" into the registry with a "name" using the put() method. (The registry then owns the object.) These object can then be retrieved using the get() method.

Note: This class is Thread-safe.
