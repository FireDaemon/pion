Detailed Description

This class is used to produce "Trace" logging. When an instance of this class is created, it will log a "ENTER: " + msg log message if TRACE_LOG_LEVEL is enabled for logger. When an instance of this class is destroyed, it will log a "ENTER: " + msg log message if TRACE_LOG_LEVEL is enabled for logger.
