Tokenize s using c as the delimiter and put the resulting tokens in _result. If collapseTokens is false, multiple adjacent delimiters will result in zero length tokens.

Example: 

  string s = // Set string with '.' as delimiters
  list<log4cplus::tstring> tokens;
  tokenize(s, '.', back_insert_iterator<list<string> >(tokens));
