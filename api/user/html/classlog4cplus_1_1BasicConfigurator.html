Detailed Description

Use this class to quickly configure the package. For file based configuration see PropertyConfigurator. BasicConfigurator automatically attaches ConsoleAppender to rootLogger, with output going to standard output, using DEBUG LogLevel value. The additional parameter logToStdErr may redirect the output to standard error.
